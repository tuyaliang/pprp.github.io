- 感受野的总结
- 调参方面：
  - 合适的学习率
  - 调参流程：
    - 从简单模型入手
    - 成功搭建模型，重现结果
    - 由粗到细随即搜索优化超参数
    - 如果欠拟合，增大模型；如果过拟合就添加数据或者调整
  - 常见错误类型：
    - tensor形状
    - 预处理输入错误
    - 损失函数错误输入
    - 忘记设置正确训练模型
    - 错误的数据类型
  - 测试集错误率评估模型：
    - 测试机错误率=错误率下限+偏移+方差+分布偏差+过拟合
  - 欠拟合解决方法：
    - 模型更大（filter+更多的层）
    - 减少正规化
    - 错误分析（效果展示）
    - 换一种模型架构
    - 超参调节（测试模型对不同超参的敏感性）--贝叶斯优化
      - learning rate
      - optimizer choice
      - batch size
      - weight initialization
      - loss function
      - model depth
    - 加入更多特征（route）
